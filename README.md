# Agens

[![Go Reference](https://pkg.go.dev/badge/github.com/gonzxlezs/agens.svg)](https://pkg.go.dev/github.com/gonzxlezs/agens)

## Overview

**agens** is a Go package designed to help you create and manage AI agents. 
It was developed as a learning project to explore the use of AI-powered applications. 
The package provides basic components and abstractions to simplify the creation of agents that can handle conversations and perform actions.

## Installation

``` bash
$ go get github.com/gonzxlezs/agens
```
